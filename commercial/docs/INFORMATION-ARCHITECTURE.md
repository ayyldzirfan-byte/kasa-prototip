# Kasam Commercial Information Architecture

## Main Tabs
1. Ana ekran
2. Hareketler
3. Bütçeler
4. Takvim
5. Rapor

## Ana Ekran
- Profile summary.
- Personal vault net position.
- Financial rhythm score.
- Today, week, month net effect.
- Recent personal-impact movements.
- Shared vault effects.

## Hareketler
- Personal-impact ledger by default.
- Filter by vault, date, type, status.
- Hidden surprise rows show locked state until revealed.
- Movement detail opens media, split, and receipt actions.

## Bütçeler
- Personal vault card.
- Shared vault cards.
- Shared vault detail:
  - Vault total.
  - My impact.
  - Members.
  - Movements.
  - Share/invite.
  - Settlement.

## Takvim
- Month view.
- Movement chips by day.
- Future income and payment visibility.
- Selected day add flow.

## Rapor
- Daily receipt.
- Weekly summary.
- Monthly report.
- Personal impact.
- Shared distribution.
- Export/share action.

## Movement Add Flow
1. Select vault.
2. Select income or expense.
3. Enter amount and currency.
4. Select date.
5. Select title/category.
6. Select participants for shared effect.
7. Add optional media.
8. Choose direct save or guess game.
9. Save with submit lock and visible progress.

## Guess Game
1. Who added it?
2. Income or expense?
3. What was it for?

The actor never sees their own pending surprise as a task.
